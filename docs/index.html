羽成公園 集会所 自治会 このサイトについて

羽成公園

観音台には２つの自治会があり、羽成公園は、この地域の憩いの場です。
公園内には、野球場やテニスコートがあり、皆さんがスポーツやレクリエーションを楽しんでいます。

公園の一角には集会所があり、両自治会のイベントや会合、地域活動の拠点となっています。

集会所

集会所は、下記の２つの自治会が共同で利用しています。
集会所の予約は、役員がサークルスクエアで行っていますが、 下のリンクからも、会員が状況を確認できるようにしています。
集会所予約状況カレンダー（このパスワードは、有志に聴く）

自治会

第１自治会

  • 親睦イベントや運営は独立
  • 集会所の利用・予約は第2自治会と共有
  • サイト管理・更新に有志で協力

第２自治会

  • 親睦イベントや運営は独立
  • 集会所の利用・予約は第1自治会と共有
  • サイト管理・更新を有志でサポート
