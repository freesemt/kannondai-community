# 見直しにおける原則

**作成日**: 2026年02月07日  
**関連**: [officer_system_discussion.md](../officer_system_discussion.md) Section 8

## 原則の設定

会費改定案解説書において、起案者（高橋氏）が考慮している（今後も継続検討する）原則:

### 原則1: 増税回避

**内容**:
すべての対象者に対して、**会費または利用料が今より増えないようにする**

**具体的には**:
- 現在の普通会員¥3,600 → 据え置き or 減額のみ
- 現在の非会員¥12,000 → 据え置き or 減額のみ
- 新設される会員区分（賛助会員など）も現行料金を超えない

**例**:
- 簡便性重視案: ¥3,000（¥600減額）+ 賛助¥6,000（新設だが¥12,000より安い）
- 多様化許容案: ¥3,600（据え置き）+ 賛助¥6,000（非会員¥12,000からの選択肢）

**背景**:
- 会員の負担感を増やさない
- 改革への抵抗を最小化する
- 「北風と太陽」アプローチ（強制ではなく選択肢の提供）

---

### 原則2: 活動維持

**内容**:
中長期の財務計画が大幅に縮小し、**自治会活動が後退しない点に配慮する**

**具体的には**:
- 収入の急激な減少を避ける
- 市の業務委託料（約¥2,400/世帯）を確保する仕組みを維持
- 新しい会員区分（賛助会員）でも実質的な収入は維持

**例（多様化許容案）**:
- 賛助会員¥6,000の論理:
  - 回覧義務により市の業務委託料¥2,400を確保
  - 実質的な会費収入: ¥6,000 - ¥2,400 = ¥3,600（普通会員と同額）
  - ゴミ清掃義務により、ゴミ集積所維持コストも負担

**シミュレーションの必要性**:
- 賛助会員が増えた場合の収入変化
- 非会員が減った場合の収入増（¥12,000 → ¥6,000移行でも財務的にプラス）
- 最悪ケース（賛助会員が過半数になった場合）の試算

**懸念事項**:
- 賛助会員が大幅に増えた場合、班長選出が困難になるリスク
- しかし財務的には問題ない（むしろ改善される可能性）

---

### 原則3: 運営支障回避

**内容**:
役員選出などの運営に、**中長期でも支障がないよう十分に考慮する**

**具体的には**:
- 班長選出の困難化を避ける（賛助会員比率の管理）
- 三役選出の柔軟性を確保（世話役制度、緩やかな輪番）
- 第6班問題などの特殊事情への対応策を用意

**懸念事項**:
1. **賛助会員増加リスク**:
   - 「¥6,000で班長免除」が魅力的すぎて、賛助会員が過半数になる可能性
   - その場合、輪番周期が短くなり、運営が困難化
   - 対策: 比率の上限設定？班長役割の最小化？

2. **第6班問題の長期化**:
   - 応急対策（高橋氏の変則的継続）は2026年度のみ
   - 2027年度以降も同様の問題が繰り返されるリスク
   - 対策: 頻度調整、賛助会員制度による非会員の移行促進

3. **世話役への依存**:
   - 特定の人（高橋氏）に依存する構造
   - 世話役が辞めたらどうするか？
   - 対策: 複数世話役の育成、役割の明文化

**長期的な対策の必要性**:
- 試行期間中の評価基準の設定
- 問題が生じた場合の改善策
- 「ダメなら戻す」ルールの明確化（論点H参照）

## 3つの原則の相互関係

### 原則間のトレードオフ

**原則1（増税回避）と原則2（活動維持）**:
- 増税しないと収入が減る可能性
- しかし、賛助会員制度により、むしろ収入増の可能性もある
- 市の業務委託料の確保が鍵

**原則2（活動維持）と原則3（運営支障回避）**:
- 活動を維持するには役員が必要
- しかし賛助会員が増えると役員選出が困難化
- バランスの取り方が課題

**原則1（増税回避）と原則3（運営支障回避）**:
- 「班長免除に高額を払わせる」ことで選出困難を回避できる
- しかし増税回避の原則に反する
- ¥6,000は妥当なバランス点か？

### 3つの原則を同時に満たすための工夫

**多様化許容案の論理**:
1. **増税回避**: ¥3,600据え置き、非会員¥12,000 → 賛助¥6,000（減額）
2. **活動維持**: 回覧義務により市の委託料確保、実質的な収入は維持
3. **運営支障回避**: 世話役制度、緩やかな輪番、第6班問題への対応

**しかし懸念も**:
- 賛助会員が過半数になったら？（原則3に抵触）
- 世話役が持続可能か？（原則3に抵触）
- 複雑すぎて理解されないのでは？（合意形成の困難）

## 原則の検証

### 検証の必要性

**注意点**:
- この原則が実際に守られるか否かは、**検証が必要**
- 特に「賛助会員が大幅に増えた場合」の財務シミュレーションが重要
- 試行期間中の評価基準として活用

### 評価指標（案）

**原則1（増税回避）の検証**:
- [ ] 各会員区分の料金設定が現行以下に収まっているか？
- [ ] 料金変更により負担が増える世帯がないか？

**原則2（活動維持）の検証**:
- [ ] 年間収入が現行水準を維持しているか？
- [ ] 市の業務委託料が確保されているか？
- [ ] 支出（活動費）が削減されていないか？

**原則3（運営支障回避）の検証**:
- [ ] 班長選出が困難化していないか？（選出にかかった時間、希望者の数）
- [ ] 三役選出が困難化していないか？
- [ ] 賛助会員比率が適正範囲に収まっているか？（例: 30%以下）
- [ ] 世話役が過度な負担を抱えていないか？

### 定期的な見直し

**提案**:
- 試行期間（1〜2年）中、半年ごとに評価
- 評価結果を役員会・総会で報告
- 必要に応じて制度の微調整
- 3つの原則が守られない場合、制度を見直す or 元に戻す

## 3つの原則の限界

### 原則だけでは解決できない問題

**第6班の非会員集中問題**:
- 3つの原則を守りながらでは、根本的な解決は困難
- 応急措置（世話役、頻度調整）は可能だが、構造的問題は残る
- 班の合併など、より抜本的な改革が必要かもしれない

**世話役の持続可能性**:
- 原則3「運営支障回避」には、世話役制度が前提
- しかし世話役が無報酬で持続可能か？（論点F参照）
- 報酬を払うと原則2「活動維持」に抵触する可能性

**合意形成の困難**:
- 原則を守っても、合意形成ができるとは限らない
- アンケート結果（保守派56% vs 革新派44%）が示す分断
- 「考え方を変えるには時間が必要」（反対派の意見）

### 原則の優先順位

**もし3つを同時に満たせない場合**:
1. **原則1（増税回避）が最優先**: 会員の負担増は避けるべき
2. **原則3（運営支障回避）が次**: 役員選出できなければ自治会が機能しない
3. **原則2（活動維持）は柔軟に**: 活動を縮小してでも運営を維持

**ただし**:
- この優先順位は提案者（高橋氏）の考え
- 会員全体の意向と一致するか不明
- 総会での議論が必要
