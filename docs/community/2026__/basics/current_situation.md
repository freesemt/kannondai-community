## 非会員の分布

| 班 | 非会員数 | 班内比率 | 備考 |
|----|---------|---------|------|
| 第1班 | 0世帯 | 0% | - |
| 第2班 | 1世帯 | 4.5% | - |
| 第3班 | 0世帯 | 0% | - |
| 第4班 | 1世帯 | 5.9% | - |
| 第5班 | 1世帯 | 5.0% | - |
| 第6班 | 7世帯 | 35.0% | **非会員が集中** |
| **合計** | 10世帯 | 8.9% | - |

### 重要な観察

- 非会員の70%（7/10世帯）が第6班に集中している
- 第6班では3世帯に1世帯以上が非会員（35%）
- 第1班と第3班には非会員がいない
- **この偏在が第6班の役員選出危機の根本原因**
