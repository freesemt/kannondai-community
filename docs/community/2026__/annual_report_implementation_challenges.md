## 基本原則：憲法と刑法の順序

**原則**：「憲法（基本原理）を先に論じるべきであって、刑法（個別対処）から入るのはよくない」

- **draft_v2**は「憲法」（3層モデル、健康な構造のビジョン）を提示
- これから必要なのは「刑法」（実装の現実的課題への対策）
- **「ここからが本当の仕事」**：理想だけでなく、現実的問題への対策が必要

## 3つの課題の相互関係

### 財政と役員の相互依存

**相互作用**：
- 役員免除が増える → 会費収入への影響
- 会費収入が減る → 役員の負担軽減（第3階層削減）が必要
- 役員負担が軽減 → 役員引受のハードルが下がる
- 役員引受が増える → 会費収入の安定

### 防衛策と信頼の関係

**相互作用**：
- 会計透明化 → 信頼向上 → 参加率向上
- 明確なルール → 不公平感の削減 → 退会防止
- 過度な監視 → 不信感 → 参加意欲低下（避けるべき）

### バランスの取り方

**推奨アプローチ**：
1. **第3階層の支出削減**で、役員負担を軽減
2. **役員引受のインセンティブ**を強化（会費差額等）
3. **世話役制度**で、三役の負担を分散
4. **会計透明化**で、信頼を獲得し参加率を向上
5. **最小限の防衛策**で、悪用を防ぎつつ信頼を維持

## 次のアクション

**3つの選択肢**：

1. **財政見通しの明確化**から始める
   - 6-15の調査を基に原価計算
   - アンケート結果から会員区分予測
   - 収支シミュレーション
   - **推奨理由**：数字が固まれば、他の課題も検討しやすい

2. **三役選出の安定性**から始める
   - 世話役制度の具体化（4-2提案）
   - 役員負担軽減策の整理
   - 第6班問題への対応
   - **推奨理由**：緊急性が高い（第6班の班長選出が迫る）

3. **両方を並行して**進める
   - 相互依存関係があるため
   - **推奨理由**：時間制約を考えると並行作業が現実的

**推奨：並行作業**
- まず**財政見通し**の数字を固める（6-15の原価データ活用）
- その数字を使って**三役選出**と**防衛策**を設計
- 防衛策は制度設計と同時に組み込む

**または**：
- まず**現状データ確認**から始める
  - 支出内訳の詳細取得
  - 会員構成の分析
  - アンケート結果の再整理
