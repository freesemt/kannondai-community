**作成日**: 2026年02月07日

## 概要

会費改定案と役員選出制度の改革について、今後検討すべき論点を整理します。

**論点の体系**:
- **論点A**: 会員区分の細分化の是非
- **論点B**: 「北風と太陽」アプローチの有効性
- **論点C**: 非会員の集中問題と使用料の見直し → [第6班危機分析](group6/crisis.md)を参照
- **論点D**: コミュニティの一体性と回覧の重要性
- **論点E**: コミュニティの一体性と班長免除
- **論点F**: 世話役制度の持続可能性
- **論点G**: 三役選出頻度の調整と慣例からの妥当性 → [第6班危機分析](group6/crisis.md)を参照
- **論点H**: 試行可能性

## 論点A: 会員区分の細分化の是非

### 論点の内容

**提案**: 4つの会員区分（普通・優遇・賛助・非会員）を導入する

| 区分 | 年額 | 班長義務 | 回覧義務 | ゴミ清掃義務 |
|------|------|---------|---------|-------------|
| 普通会員 | ¥3,600 | ○ | ○ | ○ |
| 優遇会員 | ¥3,600 | × | ○ | ○ |
| 賛助会員 | ¥6,000 | × | ○ | ○ |
| 非会員 | ¥12,000 | × | × | ゴミのみ |

### 賛成の立場

**主張**: 多様性への柔軟な対応が必要

**論拠**:
1. **多様な価値観や事情に対応できる**
   - 高齢者世帯への配慮
   - 班長を引き受けられない事情（介護、仕事など）への柔軟な対応
   - 「価値観の多様化を受け入れることが時代の要請」

2. **選択肢の提供**
   - 非会員¥12,000 → 賛助会員¥6,000への移行促進
   - 「班長は無理だが、回覧とゴミ清掃なら協力できる」という層を取り込む

3. **第6班問題の緩和**
   - 非会員7世帯のうち何世帯かが賛助会員に移行すれば、連帯感が回復する可能性
   - 役員選出の困難が緩和される

### 反対・懸念の立場

**主張**: シンプルな制度を維持すべき

**論拠**:
1. **ルールが複雑化しすぎる**
   - 4つの会員区分は理解・運用が困難
   - 「優遇会員」「賛助会員」という呼称の曖昧さ
   - 会費徴収、会員管理の煩雑化

2. **既成のルールの軽視**
   - 既成のルールが成立した事情・経緯を無視している
   - 簡単に変更すると、ルールへの信頼が失われる

3. **合意形成の困難**
   - 「多様化許容」を徹底しすぎると、共通理解が得られない
   - アンケート結果（保守派56.2% vs 革新派43.8%）が示す分断

4. **時間が必要**
   - 考え方を変えるには時間が必要
   - 拙速な導入は混乱を招く

### 判断材料として必要な情報

- [ ] 他の自治会の事例（シンプルな制度 vs 細分化された制度）
- [ ] 運用の複雑さ vs 柔軟性のトレードオフの評価
- [ ] 会員管理システムの負担増の試算
- [ ] 簡便性重視案（¥3,000+賛助¥6,000）との比較

## 論点B: 「北風と太陽」アプローチの有効性

### 論点の内容

**提案**: 強制より自発性を尊重する

**「北風と太陽」の比喩**:
- **北風**: 義務・ペナルティで強制する（現行の非会員¥12,000）
- **太陽**: 選択肢を提供し、自発的な協力を促す（賛助会員¥6,000）

**参考資料**: 「希望役員制の試行方法.pdf」(2025年08月12日)

### 賛成の立場

**主張**: 自発性が活動の質を高める

**論拠**:
1. **意欲のある人による運営**
   - 義務でやらされる人より、自発的に参加する人の方が活動の質が高い
   - コミュニティの活性化につながる

2. **現実的な対応**
   - 強制しても従わない人は従わない（非会員が増える）
   - 選択肢を提供することで、部分的な協力を引き出せる

3. **価値観の尊重**
   - 多様な価値観を認め合うことが現代社会の要請
   - 「参加したくない人を無理に参加させない」ことの重要性

### 反対・懸念の立場

**主張**: 本当に十分な希望者が集まるか不明

**疑問**:
1. **希望者の不足リスク**
   - 「誰もやりたくない」場合、どうするのか？
   - 班長選出が困難になるリスク（賛助会員が過半数になった場合）

2. **バックアッププランの欠如**
   - 集まらなかった場合の対応が不明確
   - 「試してみてダメなら戻す」は本当に可能か？（論点H参照）

3. **公平性の問題**
   - 自発的に引き受ける人に負担が集中しないか？
   - 「やりたくない人がやらない」ことの不公平感

### 判断材料として必要な情報

- [ ] 他の自治会での「希望役員制」の実施例と成否
- [ ] 試行期間の設定と評価基準
- [ ] バックアッププラン（希望者不足時の対応）の明確化

## 論点D: コミュニティの一体性と回覧の重要性

### 論点の内容

**提案の論理**:
- 「全構成員が、最低限の義務として回覧に協力し、一定の連帯感を維持する」
- 回覧義務化により市の業務委託料（約¥2,400/世帯）を確保
- 賛助会員が増えても財務的に問題ない（むしろプラス）

**会員区分と回覧義務**:
| 区分 | 回覧義務 | 備考 |
|------|---------|------|
| 普通会員 | ○ | - |
| 優遇会員 | ○ | - |
| 賛助会員 | ○ | これにより市の委託料¥2,400確保 |
| 非会員 | × | 市の委託料なし |

### 賛成の立場

**主張**: 回覧協力を最低限の参加として位置づける

**論拠**:
1. **財務的な理由**
   - 市の業務委託料（約¥2,400/世帯）の確保
   - 賛助会員¥6,000 - ¥2,400 = ¥3,600（実質的には普通会員と同額の収入）

2. **コミュニティの一体性**
   - 回覧板が回ることで、最低限のつながりを維持
   - 情報共有の仕組み

3. **非会員との差別化**
   - 「回覧協力」を会員と非会員の境界線とする
   - ¥12,000 → ¥6,000で大幅な負担軽減（非会員からの移行促進）

### 反対・懸念の立場

**疑問**: 回覧だけで「連帯感」は維持できるか？

**論拠**:
1. **デジタル時代の疑問**
   - 紙の回覧板は時代遅れでは？
   - メール、LINE、掲示板など他の方法はないか？

2. **実質的な連帯感**
   - 回覧板を回すだけで「コミュニティの一体性」が維持されるのか？
   - 形式的な義務に過ぎないのでは？

3. **非会員との分断**
   - 回覧を回さない非会員との分断は深刻化しないか？
   - 「回覧板が回らない世帯」の孤立化リスク
   - 災害時など緊急時の連絡手段は？

4. **負担感の問題**
   - 高齢者世帯や日中不在の世帯には回覧が負担
   - 回覧義務があることで、むしろ非会員化が進むリスク

### 判断材料として必要な情報

- [ ] 市の業務委託料の詳細（金額、条件、用途）
- [ ] 回覧板のデジタル化の可能性と課題
- [ ] 非会員世帯への情報伝達方法（現状と改善案）
- [ ] 他の自治会での情報共有方法の事例

## 論点E: コミュニティの一体性と班長免除

### 論点の内容

**提案**: 賛助会員（¥6,000）は班長免除

**論理**:
- 班長義務の対価: ¥2,400
- 普通会員¥3,600 - 賛助会員¥6,000の実質差額 = ¥2,400

### 賛成の立場

**主張**: 意欲のある人による運営で活性化する

**論拠**:
1. **柔軟な対応**
   - 「班長は無理だが、回覧とゴミ清掃なら協力できる」という層を取り込む
   - 高齢者、介護中、仕事多忙など、事情がある世帯への配慮

2. **役員の質の向上**
   - 意欲のある人が役員になることで、活動の質が向上
   - 「やりたくない人に無理にやらせる」ことの弊害を回避

3. **経済的選択肢**
   - ¥2,400の追加負担で班長免除
   - 時間的負担を金銭的負担に転換する選択肢

### 反対・懸念の立場

**疑問**: 「会員」と「非会員」の二層構造がもたらす分断は？

**論拠**:
1. **賛助会員の増加リスク**
   - 「¥6,000で班長免除」が魅力的すぎて、賛助会員が過半数になるリスク
   - 普通会員が減少すると、輪番周期がさらに短くなる
   - 最悪の場合、班長選出が不可能になる

2. **公平性の問題**
   - ¥2,400で班長免除を「買える」ことの公平性
   - 経済的に余裕のある世帯が班長免除を選び、余裕のない世帯が負担を強いられる構造

3. **コミュニティの分断**
   - 「班長をやる人」と「やらない人」の二層構造
   - 「やらない人」への不満や反感のリスク
   - コミュニティの一体性が損なわれる可能性

4. **非会員¥12,000との関係**
   - 回覧板も回さない「非会員」は自治会員と言えるか？
   - 賛助会員（回覧協力）と非会員の差¥6,000は妥当か？

### 判断材料として必要な情報

- [ ] 賛助会員比率の上限設定（例: 30%まで）の検討
- [ ] 班長の役割を最小化し、負担感を下げる具体策
- [ ] 財務シミュレーション（賛助会員20%, 30%, 50%のケース）
- [ ] 他の自治会での班長免除制度の事例

## 論点F: 世話役制度の持続可能性

### 論点の内容

**提案内容**:
- 有志無償の「世話役」を新設
- 中長期で自治会を支援、三役の選出・実務を支援
- 当面は提案者（高橋氏）が引き受ける

**詳細**: [役員選出方針](officer_selection_policy.md) Section 2.3を参照

### 賛成の立場

**主張**: 輪番制の弱点を補う必要がある

**論拠**:
1. **ノウハウの蓄積**
   - 輪番役員（1年任期）では運営ノウハウが蓄積しにくい
   - 世話役が継続的にサポートすることで解決

2. **中長期視点の維持**
   - 中長期的な課題（会費改定、役員選出改革など）の継続検討
   - 総会資料作成などの専門的作業の支援

3. **柔軟な対応**
   - 特殊事情（第6班危機など）への応急措置
   - 変則的な役割分担の調整

4. **実績の存在**
   - 高橋氏が実際に世話役的役割を果たしている
   - 第6班問題への対応（Nさんとの協働）で必要性が実証されている

### 反対・懸念の立場

**疑問**: 特定の人に依存する構造になっていないか？

**論拠**:
1. **持続可能性の疑問**
   - 無報酬で継続できるか？負担が大きすぎないか？
   - 「当面は高橋氏」の後、誰が引き受けるのか？
   - 有志が見つからなければ制度が崩壊する

2. **ノウハウ継承の問題**
   - 世話役が交代するときのノウハウ継承は？
   - 「特定の人にしかできない」状態になるリスク

3. **権限の曖昧さ**
   - 世話役の役割と権限が不明確
   - 「支援」と「実質的な決定」の境界線は？
   - 役員（輪番）の形骸化リスク

4. **長期的な形態の疑問**
   - 「有志無償」という形態が長期的に維持できるか？
   - 報酬を払うべきでは？（その場合の財源は？）

### 判断材料として必要な情報

- [ ] 世話役の役割と権限の明文化
- [ ] 世話役の選出方法（有志だけでいいのか？）
- [ ] 世話役の任期と交代ルール
- [ ] 複数世話役の体制構築
- [ ] 報酬化の検討（財源の試算）
- [ ] 他の自治会での類似制度（事務局、顧問など）の事例

## 論点H: 試行可能性

### 論点の内容

**主張**: まず試してみて、ダメなら戻せばいい

**「試行方法」の提案**:
- 一定期間（1〜2年）試行する
- 評価基準を設定し、成否を判断
- ダメなら元に戻す

### 賛成の立場

**主張**: 実験的アプローチが合理的

**論拠**:
1. **理論だけでは分からない**
   - 実際にやってみないと分からないことが多い
   - 予想外の効果（良い意味でも悪い意味でも）が出る可能性

2. **リスク管理**
   - 試行期間を設定することで、リスクを限定
   - 失敗しても被害を最小化できる

3. **学習機会**
   - 試行を通じて、改善点が明確になる
   - 会員全体も制度を理解する機会になる

### 反対・懸念の立場

**疑問**: 一度導入した制度を戻すのは容易か？

**論拠**:
1. **「戻す」ことの困難さ**
   - 一度変更した制度を元に戻すのは、政治的・心理的に困難
   - 「試行」のつもりが「既成事実化」するリスク
   - 賛助会員になった人が、元に戻ることを拒否する可能性

2. **試行期間中の混乱**
   - 混乱をどう最小化するか？
   - 「試行中」という不安定な状態が続くことの弊害

3. **評価基準の曖昧さ**
   - 何をもって成功/失敗と判断するか？
   - 判断基準が曖昧だと、「戻す」判断ができない

4. **2つの制度の並行運用**
   - 「現状」と「試行」を並行運用するのか？
   - それは現実的か？（事務負担、会員の混乱）

### 判断材料として必要な情報

- [ ] 試行期間の具体的設定（1年？2年？）
- [ ] 評価基準の明確化（具体的な数値目標）
  - 役員選出困難度（希望者の数、選出にかかった時間など）
  - 会員満足度（アンケート）
  - 財務状況（収入、支出、安定性）
  - 賛助会員比率
- [ ] 復帰ルール（一度「賛助会員」を選んだ人が元に戻る際のルール）
- [ ] 試行期間中の評価方法（定期的なアンケート、役員会での検討など）
- [ ] 「戻す」判断の手続き（総会決議が必要？）
