# 自治会の簡単なモデル

**作成日**: 2026年2月7日  
**目的**: 現実の問題を単純化したモデルで精密な議論を可能にする

---

## 1. モデルの目的

複雑な現実の自治会問題をそのまま解くのは難しいため、前提を単純化して精密な議論をしやすくする。

**適用予定の問題**:
1. **ゴミ集積所利用料12,000円の違法性証明**（本文書の主目的）
2. その他の問題（今後追加予定）

## 2. 自治会の目的の階層構造

自治会で達成したいことには階層がある：

### レベル1: 基本的生活権（最低限）
- **目的**: 普通に生きていくための最低限の生活を守る
- **具体例**: ゴミが捨てられる
- **性質**: 基本的人権に近い、生存権レベル
- **選択性**: 選択の余地なし（すべての人に必要）

### レベル2: 安全と安心（中位）
- **目的**: できるだけ安全なところで暮らしたい
- **具体例**: 防犯、防災、見守り活動など
- **性質**: 生活の質の向上
- **選択性**: 一定の選択の余地あり

### レベル3: 充実した生活（上位）
- **目的**: 気持ちのよい隣人の中で楽しく暮らしたい
- **具体例**: 交流イベント、コミュニティ活動
- **性質**: 生活の豊かさ、幸福追求
- **選択性**: 個人の価値観による選択

### 階層の意味

- **下位ほど普遍的**: レベル1は誰にでも必要、レベル3は価値観による
- **上位ほど多様性**: レベル3では「楽しい」の定義が人により異なる
- **費用負担の正当性**: 下位レベルは強制負担が正当化されやすい、上位レベルは困難

## 3. 考え方の押し付けの問題

### 3.1 問題の本質

**問い**: レベル1しか期待しない人にレベル3を押し付けていないか？

### 3.2 具体的な問題

**現状の自治会運営**:
- すべての会員に一律の会費（¥3,600）と役員義務を課す
- しかし人によって自治会に期待するレベルは異なる：
  - **A氏**: レベル1のみ希望（ゴミが捨てられればよい）
  - **B氏**: レベル1-2希望（ゴミ＋安全）
  - **C氏**: レベル1-3希望（ゴミ＋安全＋交流イベント）

**押し付けの構造**:
- A氏（レベル1だけ希望）にも、レベル3の活動費用と役員義務を強制
- → A氏の選択の自由を侵害
- → A氏は「非会員」を選択せざるを得ない（¥12,000/年）

### 3.3 なぜ問題なのか

1. **価値観の多様性の無視**:
   - 「楽しい交流」が全員にとって価値があるとは限らない
   - レベル3は個人の価値観による選択であるべき

2. **基本的権利への不当な条件付け**:
   - レベル1（基本的生活権）を得るために
   - レベル3（価値観による選択肢）への参加を強制
   - → 不当な条件付け

3. **強制的な会費・役員義務**:
   - A氏はレベル3に価値を感じないのに費用負担
   - レベル3の活動を運営する役員義務も負わされる

### 3.4 「非会員」という選択の意味

**現状**: レベル1だけ希望する人の選択肢は「非会員」（¥12,000/年）のみ
- これは実質的に「レベル3を拒否したことへの懲罰」
- レベル1の基本的権利に不当に高額な料金を設定

## 4. ゴミ集積所利用料12,000円の問題

### 3.1 現状

**非会員のゴミ集積所利用料**: 年額12,000円
- 現行制度で徴収
- 会費改定案の全選択肢（選択肢1,2,3）でも維持

### 4.2 実際の費用試算

**ゴミ集積所の実際の管理費**:

**設備費用**:
- 集積ボックス: 10個
- 1個あたりの交換費用: ¥250,000
- 合計交換費用: ¥2,500,000
- 耐用年数: 30年

**年間費用の計算**:
```
年間設備償却費 = ¥2,500,000 ÷ 30年 = ¥83,333/年
```

**1世帯あたりの費用**:
```
総世帯数 = 112戸
1世帯あたり年間費用 = ¥83,333 ÷ 112戸 ≒ ¥744/年
```

**その他の費用**（清掃等）を考慮しても、1世帯あたり¥1,000〜2,000/年程度と推定される。

### 4.3 違法性の論点

**問題の本質**:
- **実際の費用**: 約¥744〜2,000/年
- **非会員への請求額**: ¥12,000/年
- **差額**: 約¥10,000〜11,000/年（**実費の6〜16倍**）

**違法性の根拠**:

1. **基本的生活権の侵害**:
   - レベル1（ゴミが捨てられる）= 最低限の生活権
   - これに対して実費の6〜16倍の高額料金を徴収
   - → 基本的生活権を人質にした不当な料金設定

2. **懲罰的料金設定**:
   - 実費: ¥744〜2,000/年
   - 請求額: ¥12,000/年
   - → 差額¥10,000は実費負担ではなく「レベル3を拒否したことへの懲罰」
   - → 自治会への不参加を事実上不可能にする

3. **選択の自由の侵害**:
   - レベル1だけ希望する人（A氏）の合理的な選択肢がない
   - 会員（¥3,600 + 役員義務）: レベル3を強制される
   - 非会員（¥12,000）: 懲罰的な高額料金
   - → 実質的に選択の自由がない

4. **営利事業化の禁止**:
   - 自治会は非営利組織
   - ゴミ集積所の運営は営利事業ではない
   - 実費を大幅に超える料金徴収 = 事実上の営利行為
   - → 自治会の性質に反する

### 4.4 適正な料金設定

**原則**: 基本的生活権（レベル1）に関わるサービスは実費負担が限度

**適正な非会員料金の試算**:
- 実費: ¥744〜2,000/年
- 管理リスク等を考慮しても: ¥2,000〜3,000/年程度が上限
- 現行¥12,000は明らかに不当

### 4.5 関連判例（参考）

**福井地裁判決（2024年4月16日）**

- **事案**: 自治会退会者のゴミステーション使用権を巡る訴訟
- **判決**: 年15,000円を支払うことを条件に使用権を認める
- **裁判所の論理**:
  - ゴミステーション管理費だけでなく「町内会活動を存続・維持するための費用」も考慮
  - 町内会の活動経費（約186万円）から市補助金を除き、世帯数で割った額
- **参照**: [福井新聞記事](https://www.fukuishimbun.co.jp/articles/-/2288722)

**注意**: 各自治会の事情は異なるため、この判例を短絡的に適用することはできない。参考情報として記録。

### 5.5 構造的問題の所在

#### 5.4.1 倫理的な核心問題

**根本的な問い**：
> レベル3を望む人が、レベル1しか望まない人に、何かを押し付けていいのか？

#### 5.4.2 現行制度の構造

**会員の場合**（¥3,600 + 役員義務）:
- レベル1〜3のすべてに参加（強制）
- 集会所を使う権利
- サークル活動に参加する権利
- しかし、レベル1だけでいい人（A氏）にとっては不要な権利

**非会員の場合**（¥12,000）:
- レベル1のみ利用（ゴミ集積所）
- レベル2-3は利用しない
- しかし¥12,000を支払う

#### 5.4.3 ¥12,000の内訳分析

**レベル1の実費**:
- ゴミ集積所: ¥744〜2,744/年
- これだけがレベル1（基本的生活権）に対応

**差額の正体**:
- ¥12,000 - ¥744〜2,744 = **¥9,256〜11,256**
- この差額は何の費用なのか？

**レベル2-3の費用**（会員が負担している費用）:
- 集会所（改築）: ¥1,929〜3,214
- 集会所（エアコン）: ¥193
- 集会所（年間経費）: ¥244
- 自治会運営費: ¥446
- 役員手当（レベル2-3分）: 約¥1,500〜1,700（推定）
- その他（回覧等）: ¥1,000〜2,000
- **小計: ¥5,312〜8,797/年**

**結論**: 
- 非会員の¥12,000には、レベル1の実費（¥744〜2,744）に加えて
- レベル2-3の費用（約¥5,000〜8,000）が**上乗せされている**
- → **レベル3を望む人が、レベル1だけの人に、レベル2-3の費用を押し付けている**

#### 5.4.4 集会所の必要性に関する倫理的考察

**レベル1だけ希望する人（A氏）の視点**:

1. **集会所は不要**:
   - ゴミが捨てられればよい
   - 交流イベントに興味がない
   - 集会所の維持に¥2,366〜3,651/年を払う理由がない

2. **会議ならオンラインで可能**:
   - 必要な連絡はオンラインで十分
   - 今の時代、物理的な集会所は必須ではない
   - Zoom, Teams, LINE等で代替可能

3. **押し付けられている費用**:
   - 集会所の改築: ¥1,929〜3,214/年 → A氏には不要
   - 集会所のエアコン: ¥193/年 → A氏には不要
   - 集会所の年間経費: ¥244/年 → A氏には不要
   - サークル助成金: ¥179/年 → A氏には不要
   - 役員手当（レベル2-3分）: 約¥1,500〜1,700/年 → A氏には不要（最小限の組織費¥200〜500で十分）
   - **合計: 約¥4,000〜5,500/年が押し付け**

**レベル3を望む人（C氏）の視点**:

1. **集会所は必要**:
   - 対面での交流に価値を感じる
   - サークル活動の拠点として重要
   - コミュニティの維持に必要

2. **しかし**:
   - それはC氏の**個人的な価値観**
   - A氏に同じ価値観を強制する権利はない

#### 5.4.5 倫理的判断

**原則1: 基本的権利の保障**
- レベル1（ゴミが捨てられる）= 基本的生活権
- 実費（¥744〜2,744）での提供が倫理的に要求される

**原則2: 価値観の強制禁止**
- レベル3（楽しい交流）= 個人の価値観による選択
- A氏にC氏の価値観を強制してはならない
- 集会所の費用をA氏に負担させるのは倫理的に不当

**原則3: 適正な費用負担**
- レベル1希望者: レベル1の実費のみ負担
- レベル3希望者: レベル1 + レベル2-3の費用を負担
- **各自が希望するレベルの費用だけを負担すべき**

**結論**:
> ¥12,000という料金設定は、レベル3を望む人が、レベル1だけの人に、自分たちの価値観と費用を押し付けている構造である。これは倫理的に不当である。

#### 5.4.6 福井地裁判決の限界

福井地裁判決（2024年4月16日）は：
- 「町内会活動を存続・維持するための費用も考慮する必要がある」と判断
- 年¥15,000を認めた

**しかしこの判決は**:
1. **階層構造を考慮していない**:
   - 「町内会活動」をひとまとめにしている
   - レベル1とレベル2-3を区別していない

2. **価値観の強制を容認**:
   - レベル1だけ希望する人に
   - レベル2-3の費用負担を強制することを認めた

3. **倫理的考察の欠如**:
   - 「存続・維持」の必要性を前提としている
   - しかし「誰にとって必要か」を問うていない

**本モデルが明らかにした点**:
- 町内会活動には階層がある
- レベル1だけ希望する人から、レベル2-3の費用を徴収するのは倫理的に不当
- 判決は、この構造的問題を見落としている

#### 5.4.7 適正な料金設定

**役員手当の配分問題**:
- 役員手当（¥1,875/年）は組織運営コスト
- しかし、現在の役員は主に**レベル2-3の活動**（会議、イベント、交流）に従事
- レベル1（ゴミ集積所）だけなら、必要な組織労力は最小限
  - ボックス交換の判断・見積・発注
  - 簡単な相談・調整
  - → 役員手当全額（¥1,875）は不要、推定¥200〜500/年程度

**倫理的原則に基づく料金**:

| 希望レベル | 提供サービス | 適正料金（円/年） | 内訳 |
|----------|------------|----------------|------|
| レベル1のみ | ゴミ集積所 | **2,000〜3,500** | 実費¥744〜2,744 + 最小限組織費¥200〜500 |
| レベル1-2 | ゴミ + 安全・回覧 | 3,000〜5,500 | レベル1 + 回覧等¥1,000〜2,000 |
| レベル1-3 | ゴミ + 安全 + 交流 | 6,431〜10,716 | 全費用 |

**現行との比較**:
- 現行非会員料金: ¥12,000
- 適正料金（レベル1のみ）: **¥2,000〜3,500**
- **過剰請求額: ¥8,500〜10,000**

**推奨**:
- レベル1希望者への料金: **¥2,500〜3,000/年**
- 現行¥12,000は、適正額の**4〜5倍**であり、倫理的に不当

## 7. 改革の経緯

### 7.1 現行制度の成立

**非会員料金¥12,000の設定**:
- いつ、誰が、どういう認識で決めたか: **不明**
- レベル構造の認識: **なかった**可能性が高い
- 推測される経緯:
  - 会員の実質負担（会費¥3,600 + 市委託料¥2,400 = ¥6,000）からの逆算？
  - 「自治会全体の維持費」を非会員にも負担させる発想
  - 「ただ乗り防止」の懲罰的料金
  - **根本的問題**: 「どういう自治会にしたいのか」という肝心の点があいまいなまま会則が決定された

### 7.2 第1段階の改革提案（2025年12月）

**会費改定案アンケート**:
- 実施: 2025年12月21日～2026年1月25日
- 説明資料: 会費改定案解説書（高橋作成）
- 3つの選択肢:
  1. 現状維持
  2. 簡便性重視
  3. **多様化許容**（改革案）

**多様化許容案の特徴**:
- 普通会員: ¥3,600（班長義務あり）
- **賛助会員: ¥6,000**（回覧協力、班長免除）← **新設**
- 非会員: ¥12,000（変更なし）

**戦略的判断**:
- ¥12,000の根本的見直し（→¥2,500程度）は避けた
- 理由: 一度に全部変えると合意形成が困難
- まず**中間選択肢¥6,000を設けて救済の道**を開く
- 賛助会員により「価値観の多様化を受け入れる」実践
- ¥12,000の根本的見直しは次のステップに残す

**アンケート結果**:
- 現状維持: 27票
- 簡便性重視: 23票
- 多様化許容: 39票（最多だが過半数未満）
- → 集約: 保守46票 vs 革新39票
- **決定**: 改定延期（2026年度に持ち越し）

**第1段階の限界**:
- レベル構造の明示: なし
- ゴミ集積所の実費計算: なし
- ¥12,000の倫理的問題: 踏み込まず
- 会員への説明が不十分 → 合意形成に至らず

### 7.3 第2段階の理論構築（2026年2月～）

**本モデルの構築**:
- 第1段階で踏み込まなかった根本的問題を明確化
- レベル1-3の階層構造モデルを確立
- ゴミ集積所の実費: ¥744〜2,744を計算
- 適正料金: ¥2,000〜3,500を理論的に導出
- **倫理的問題の明示**: レベル3を望む人が、レベル1だけの人に費用を押し付けている

**理論武装の完成**:
- 現行¥12,000の不当性を実費計算で証明
- 福井地裁判決の限界を指摘（階層構造の欠如）
- 次の改革提案の基盤を構築

**次のステップ**:
- このモデルを活用した改革提案
- 年報第1部での説明
- 3/01談話会での議論
- 2026年度総会での再提案
