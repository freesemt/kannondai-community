思想的検証のページ

🧠 はじめに

自治会のような（国や市に比べれば）比較的小さな組織の運営であっても、よく考えられた思想に基づいて規則を作ることが必要だと考えられます。 その意味で、一律輪番役員制で回すだけでよいのか、逆に希望役員制にすれば何が問題か、などについての検証を試みます。

標語的にいうと、「一律の役員義務から、多様な責任分担へ」という方向の可能性を探ってみようと思います。 とはいえ、過ぎた多様性の許容は崩壊に向かう懸念もあるので、適度なバランスが何かを再考してみようという意味です。
